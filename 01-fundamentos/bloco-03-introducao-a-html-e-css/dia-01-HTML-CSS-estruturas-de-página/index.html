Estes serão os próximos conteúdos que estudarei em seguida:

  • CSS
  • HTML semântico
  • JavaScript
  • DOM

Sobre os próximos Aprendizados:

  • CSS: trata-se de uma linguagem de estilização responsável por alterar a apresentação visual dos elementos de um documento web.

  • HTML semântico: veremos mais sobre tags semânticas que são auxiliam ferramentas de acessibilidade a navegar o conteúdo e o ranqueamento da página em motores de busca.

  • JavaScript: linguagem de programação interpretada de alto nível que junto do HTML e CSS é uma das principais tecnologias do desenvolvimento web.

  • DOM: trata-se da modelagem de todo o documento HTML (document object model).
